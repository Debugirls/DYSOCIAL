# DYSOCIAL

### Instalar y ejecutar:

#### Backend:

##### Abrir una terminal nueva:

```sh
cd back 
```

```sh
npm install 
```

```sh
node server.js 
```

#### Frontend:

##### Abrir otra terminal diferente:

```sh
cd front 
```

```sh
npm install 
```

```sh
npm run serve
```


### Pruebas backend postman:
